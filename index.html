Massively Scalable

Built-in Gossip and Raft protocol into the core, Swasd is massively scalable to multi-thousand nodes by design. It aims for both large-scale Cloud and IoT deployment.

Fast

Extended from the code base of Containerd, Swasd uses the same container technology that has already empowered millions of Docker Engine.

Open Standard

Based on the Open Container Initiative standard, Swasd is the first clustering technology that runs container images in the Open Container Format, which is also compatible with Docker.
